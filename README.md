# 我的开源学习仓库

## 对开源项目的贡献想法

目前还在学习阶段，计划先通过阅读优秀开源项目的源码和文档来熟悉开源社区的协作规范。本学期希望能从修复文档错误、编写测试用例等简单的入门任务开始，逐步参与到实际项目中，为开源社区贡献自己的一份力量。

## 本学期课程目标

1. 熟练掌握 Git 和 GitHub 的基本操作
2. 完成一次开源项目的 Pull Request 并被合并
3. 学习团队协作开发流程，提升代码规范意识
4. 期末能独立维护一个小型开源项目
